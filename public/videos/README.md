# Videos folder

Put your ritual clip here as **`ritual.mp4`** — a short **MP4 (H.264)**, ideally 5–10 seconds
and under ~50 MB.

Then either:
- set `videoSrc="/videos/ritual.mp4"` on the **"Devotion in motion"** `<VideoShowcase>` in
  `src/app/page.tsx`, then commit + push; **or**
- just tell Claude **"video added"** and it will wire it in and deploy.
